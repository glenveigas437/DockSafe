About DockSafe

Container Image Vulnerability Scanner with Reporting

Project Overview

DockSafe is an automated vulnerability scanner designed to secure container images before they reach production. With the rise in containerized applications, ensuring the security of container images is critical to prevent vulnerabilities from reaching production environments.

This project addresses the common problem where DevOps teams lack automated tools that integrate seamlessly into their CI/CD pipelines, resulting in manual checks or no security checks at all.

Key Goals
  • Automated vulnerability scanning
  • CI/CD pipeline integration
  • Real-time notifications
  • Historical vulnerability tracking
Technologies Used
  • Python (Flask)
  • Trivy Scanner
  • SQLite Database
  • Docker Integration
Features
Vulnerability Scanning

Automatically scan Docker images for known vulnerabilities using Trivy scanner.

Dashboard & Analytics

Visual dashboard showing scan statistics and vulnerability trends over time.

Report Generation

Generate detailed vulnerability reports with filtering and export capabilities.

Notifications

Real-time notifications via Slack, Teams, and email for critical vulnerabilities.

CI/CD Integration

Seamless integration with Jenkins, GitHub Actions, and GitLab CI pipelines.

Historical Tracking

Track vulnerability history and monitor security improvements over time.

How to Use
1
Scan Images

Use the Scanner page to scan your Docker images for vulnerabilities. Enter the image name and tag, then click "Start Scan".

2
Review Results

Check the Dashboard for scan statistics and recent activity. View detailed vulnerability information and recommendations.

3
Generate Reports

Create detailed reports from the Reports page. Filter by date, severity, and status. Export for sharing.

CI/CD Integration

DockSafe integrates seamlessly with popular CI/CD platforms to automatically scan images during the build process and prevent vulnerable images from reaching production.

GitHub Actions

Use the provided workflow template to integrate scanning into your GitHub Actions pipeline.

Jenkins

Integrate with Jenkins using the provided Jenkinsfile template for automated scanning.

GitLab CI

Use GitLab CI/CD with the provided configuration for automated vulnerability scanning.
